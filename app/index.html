aloCrafty.js

Quit messing up your perfectly formatted javascript code with html

Un poco de historia

aloCrafty nace de una de las muchas conversaciones sobre software que ocurren en la cueva.
Una tarde Xavi y Jesús estaban hablando sobre lo lamentable que es, desde el punto de vista de la la claridad en el código, escribir javascript dentro de una página html. Si bien esto es algo más o menos aceptado por casi todo el mundo, la solución que habitualmente se adopta no deja de ser igual de lamentable... ¡Insertemos el html dentro del javascritp!
Como tambien suele ocurrir en estos casos, la conversación se zanja programando.
Nos sentamos delante de ordenador y, como si de un coding dojo se tratase, empezamos a escribir tests y con rojo-verde-refactor, este DSL empezó a tomar cuerpo

//create an empty object and populate it
var a = {};

Snippets

HTML

<div class="div-test">
    <p>Chanante ipsum dolor sit amet, ahí va qué chorrazo, soooy crossoverr ullamco tontaco sed exercitation ea labore. Ad quis minim. Pero qué pelazo, ex ut tunante agazapao nostrud eiusmod artista et payacho adipisicing minim.</p>
</div>

aloCrafty.js

a.div({class:'div-test'},
  a.p('Chanante ipsum dolor sit amet, ahí va qué chorrazo, soooy crossoverr ullamco tontaco sed exercitation ea labore. Ad quis minim. Pero qué pelazo, ex ut tunante agazapao nostrud eiusmod artista et payacho adipisicing minim.')
);

HTML

<form role="form">
    <div class="form-group">
        <label for="email">Username or Email</label>
        <input type="email" class="form-control" id="email" placeholder="Enter email">
    </div>
    <div class="form-group">
        <label for="pass">Password</label>
        <input type="password" class="form-control" id="pass" placeholder="Password">
    </div>
    <button type="submit" class="btn btn-sm btn-default">Sign in</button>
</form>

aloCrafty.js

a.form({role:'form'},
  a.div({class:'form-group'},
    a.label({for:'email'},'Username or Email'),
    a.input({type:'email', class:'form-control', id:'email', placeholder:'Enter email'})
  ),
  a.div({class:'form-group'},
    a.label({for:'pass'},'Password'),
    a.input({type:'password', class:'form-control', id:'pass', placeholder:'Password'})
  ),
  a.button({type:'submit' class:'btn btn-sm btn-default'},'Sign in')
);
